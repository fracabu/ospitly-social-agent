## :gb: English

### Overview

Comprehensive social media strategy, editorial calendar, and ready-to-publish content for launching Ospitly.it on Facebook and Instagram. 80+ planned posts with visual guidelines and brand design system.

### Package Contents

| Document | Description |
|----------|-------------|
| `social-media-strategy.md` | Market analysis, personas, 12-month roadmap |
| `editorial-calendar-month1.md` | 80 planned posts for first month |
| `content-package.md` | 12 ready-to-publish content pieces |
| `page-setup-guide.md` | Facebook & Instagram setup |
| `visual-identity-guidelines.md` | Complete brand design system |

### Features

- **Market Analysis** - Italian hospitality market (€87.79B, 2025)
- **4 Target Personas** - Detailed pain points and goals
- **5 Content Pillars** - Education, Product, Success Stories, Community, Brand
- **80 Planned Posts** - Facebook + Instagram with copy and visuals
- **Brand Guidelines** - Colors, typography, templates

### Month 1 Goals

- Facebook: 500 followers
- Instagram: 300 followers
- Engagement Rate: FB 1.5%+, IG 3.5%+
- Combined Reach: 50K/month

### Quick Start

```bash
git clone https://github.com/fracabu/ospitly-social-agent.git
cd ospitly-social-agent
python -m http.server 8080
# Open http://localhost:8080/viewer.html
```
